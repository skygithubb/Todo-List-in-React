# TodoList App

This is a simple **Todo List** application built using **React**. The app allows users to add, edit, and delete tasks from their list, helping them stay organized.

## Features

- **Add Todos**: Users can add new tasks to their todo list.
- **Edit Todos**: Users can edit existing tasks.
- **Delete Todos**: Users can remove tasks from the list when they're done.
- **Responsive Design**: The app is designed to be fully responsive across different screen sizes.
